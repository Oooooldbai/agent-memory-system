# 日报格式规范 v2.0

> 金融产品日报标准格式 - 确认于 2026-04-14

---

## 9字段标准格式

每个产品必须包含以下字段：

| 字段 | 标记 | 内容要求 |
|------|------|---------|
| 序号名称 | 【N】 | 【序号】**产品名称** \| 产品类型 |
| 基本信息 | 📦 | 产品形式 + 来源（平台/机构） |
| 一句话功能 | 🔍 | 一句话描述核心价值（不超过30字） |
| 面向场景 | 👤 | 具体应用场景/目标客户 |
| 主要覆盖行业 | 🏭 | 行业范围（2-4个） |
| 代表企业 | 🏢 | 标杆客户/合作方 |
| 地区 | 🌍 | 境内/境外 + 具体区域 |
| 核心亮点 | 💡 | bullet list 3-4条，突出差异化 |
| 价值判断 | 📈 | 推荐指数 + 关联度 + 建议行动 |

---

## 示例

```markdown
【1】**中信银行 - 信e链** | 应收账款融资
📦 基本信息：产品形式：应收账款线上化融资 | 来源：中信银行官网
🔍 一句话功能：为核心企业上游供应商提供应收账款在线融资，实现T+0到账
👤 面向场景：上游供应商应收账款变现、无抵押融资、快速回笼资金
🏭 主要覆盖行业：电子制造、汽车零部件、建筑建材
🏢 代表企业：华为供应链、比亚迪供应链
🌍 地区：境内（全国）
💡 核心亮点：
- 无需抵押担保，核心企业确权即可融资
- 全程线上化，最快当天到账
- 额度随借随还，利率4-6%
- 支持多级供应商穿透
📈 价值判断：
- 推荐指数：⭐⭐⭐⭐⭐
- 关联度：高
- 建议行动：🎯 重点关注，中信银行拳头产品，市场推广成熟
```

## 输出结构

### 标题
```
💰 金融产品日报 | YYYY年MM月DD日
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 境内产品（5-7个）
【1】-【7】按9字段格式

### 境外产品（4个）
【8】-【11】按9字段格式

**境外产品必须包含**:
- HSBC / 汇丰相关产品
- Citi / 花旗相关产品
- JPMorgan / 摩根大通相关产品
- Standard Chartered / 渣打相关产品

### 今日重点推荐
选择1-2个最有价值的产品，详细说明推荐理由（3-5条）。

### 趋势观察
3-5条行业趋势总结。

### 统计信息
```
📊 今日统计：N个重点产品（境内X + 境外Y）
🔍 重点关注：[产品1]、[产品2]
💡 行业覆盖：[行业1]、[行业2]、[行业3]
```

## 筛选规则

### ✅ 包含（优先报道）
- 供应链金融（SCF）
- 产业金融
- 消费金融
- 应收账款融资
- 仓单质押融资
- 贸易融资
- 可持续供应链金融

### ❌ 排除（禁止报道）
- 股票、证券
- 大宗商品期货
- 传统银行存款/贷款
- 个人理财
- 保险
